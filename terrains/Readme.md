This folder is for different terrains the community came up with. You're welcome to contribute.  
The format consists of two files:
```
<terrain_name>.txt
<terrain_name>.json
```

The txt file is to describe what the terrain is for. What mods it suggests to be played with, and any other relevant info.  
The json file is the actual terrain file that is going to be used by the mod.

The naming convention is as follows:
```
<terrain_name>.txt
<terrain_name>_terrain.json
```
For example:
```
my_awesome_changes.txt
my_awesome_changes_terrain.json
```
